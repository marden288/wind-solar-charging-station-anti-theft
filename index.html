About the Project

A comprehensive hybrid renewable energy charging station with integrated security

🌬️ Wind Energy System

A small-scale wind turbine converts kinetic energy into AC electricity, rectified to DC. Output follows P = ½ρAv³. Powers the battery bank when solar input is low.

☀️ Solar Energy System

Polycrystalline solar panels (~17% efficiency) convert photon energy to DC power via the photovoltaic effect. MPPT algorithm ensures optimal energy harvest.

⚡ Hybrid Charging

Both sources feed into a dual-input charge controller managing priority, load balancing, and overcharge protection. Step-down converter provides regulated 5V/12V output.

🔒 Anti-Theft System

IR proximity sensor detects unauthorized access. Arduino triggers SIM800L GSM to send SMS alert. Relay module physically locks the charging port. All events are logged.

Mission & Objectives
01
Design and implement a hybrid wind-solar charging station capable of delivering stable power output under varying environmental conditions.
02
Develop an anti-theft protection system using IR sensing, relay-controlled locking, and automated SMS alerts via GSM module.
03
Build a real-time IoT monitoring platform that displays battery voltage, energy input from each source, and system health status.
04
Create a secure web application with role-based access control for administrators to manage the system remotely.
05
Demonstrate the viability of off-grid renewable energy charging as a sustainable and scalable alternative to conventional power sources.
06
Log all security events and energy metrics to a persistent database for historical analysis and performance optimization.
